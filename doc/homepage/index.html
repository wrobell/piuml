piUML — textual language for UML

About

piUML is textual language for UML. The piUML project provides language specification, language interpreter and software for diagrams generation in UML notation.

User Manual introduces into piUML, explains how to describe UML diagrams in piUML language and contains piUML language reference.

Hacking Guide is written for those interested in piUML internals and development.

piUML is free software licensed under terms of GPL license.

Features

piUML language is simple and intuitive, excellent for exchaning models of software and business processes in written communication (e-mail, messaging, source code documentation, literate programming, etc.). At the moment, the language allows to describe class, use case, component, deployment and profile diagrams. UML features like packaging, stereotypes and stereotype attributes are fully supported.

Provided piUML language interpreter is UML semantics aware (i.e. XMI or code generation could be easily supported). The interpreter can be easily embedded within 3rd party applications.

Generated diagrams are UML 2.2 notation compliant. The output is saved in high quality, vector formats like PDF, SVG or PostScript using TeX and MetaPost.

The project provides Sphinx integration.
