About

piUML is textual language for UML and software to generate diagrams in UML notation.

piUML reads UML diagram description written in piUML language and creates image containing diagram compliant with UML notation. The layout of a diagram is calculated automatically.

piUML is free software licensed under terms of GPL license.

Documentation

User Manual introduces into piUML, explains how to describe UML diagrams in piUML language and contains piUML language reference.

Hacking Guide is written for those interested in piUML internals and development.

UML specification may be downloaded from OMG specification page.

Language Features

  • Simple and intuitive language for UML.
  • Class, use cases, component, deployment and profile diagrams available. UML features like packaging, stereotypes and stereotype attributes are supported.
  • UML semantics aware - piUML is not only diagramming software. XMI could be easily supported.
  • Easy to embed within 3rd party application.

Diagramming Features

  • UML 2.2 notation compliant.
  • High quality, vector output in PDF, SVG or PostScript using Metapost.
  • Sphinx integration.
